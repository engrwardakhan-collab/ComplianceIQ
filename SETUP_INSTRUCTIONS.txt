CONTRACT ANALYZER - SETUP INSTRUCTIONS
================================================================================

QUICK START (5 MINUTES)
================================================================================

1. INSTALL NODE.JS
   - Visit: https://nodejs.org (LTS version)
   - Download and install
   - Verify: node --version (should show v18+)

2. EXTRACT THIS ZIP FILE
   - Unzip to your desired location
   - This creates a "contract-analyzer-nextjs" folder

3. OPEN TERMINAL/COMMAND PROMPT
   - Navigate to the extracted folder
   - Windows: Open command prompt in folder
   - Mac/Linux: Terminal → cd /path/to/folder

4. INSTALL DEPENDENCIES
   - Type: npm install
   - Wait for completion (2-3 minutes)

5. SET UP ENVIRONMENT
   - Copy: .env.local.example → .env.local
   - Edit: .env.local in your text editor
   - Add your OpenAI API key (get from https://platform.openai.com/api-keys)
   - File should look like:
     OPENAI_API_KEY=sk-your-actual-key-here

6. RUN LOCALLY
   - Type: npm run dev
   - Visit: http://localhost:3000
   - You should see the app!

7. TEST IT
   - Enter your OpenAI API key in the app
   - Upload a .txt contract file
   - Verify extraction works
   - Ask a question in chat

================================================================================
DEPLOY TO PRODUCTION (OPTIONAL)
================================================================================

DEPLOY TO VERCEL (RECOMMENDED - 2 MINUTES)

1. Push to GitHub
   - git init
   - git add .
   - git commit -m "initial"
   - git branch -M main
   - git remote add origin https://github.com/YOUR_NAME/contract-analyzer
   - git push -u origin main

2. Go to Vercel
   - Visit: https://vercel.com
   - Click "New Project"
   - Select your GitHub repository
   - Click "Import"

3. Set Environment Variable
   - In Vercel dashboard → Settings
   - Environment Variables
   - Add: OPENAI_API_KEY = your actual key
   - Click "Save"

4. Deploy
   - Click "Deploy"
   - Wait 2 minutes
   - Your app is live!

DEPLOY TO RENDER (ALTERNATIVE - 5 MINUTES)

1. Push to GitHub (same as above)

2. Go to Render
   - Visit: https://render.com
   - Sign up or login
   - Click "New +"
   - Select "Web Service"

3. Connect Repository
   - Connect GitHub
   - Select your contract-analyzer repo

4. Configure
   - Name: contract-analyzer
   - Environment: Node
   - Build Command: npm run build
   - Start Command: npm run start

5. Set Environment Variable
   - Scroll to "Environment"
   - Key: OPENAI_API_KEY
   - Value: your actual key
   - Click "Add"

6. Deploy
   - Click "Create Web Service"
   - Wait 3-5 minutes
   - Your app is live!

TROUBLESHOOTING
================================================================================

PROBLEM: "npm: command not found"
SOLUTION: Install Node.js from https://nodejs.org

PROBLEM: "Cannot find module 'openai'"
SOLUTION: Run: npm install openai

PROBLEM: "OPENAI_API_KEY not found"
SOLUTION: 
  1. Check .env.local exists in root folder
  2. Check it has: OPENAI_API_KEY=sk-...
  3. Restart dev server: npm run dev

PROBLEM: "Nothing happens when I upload"
SOLUTION:
  1. Check browser console (F12) for errors
  2. Verify API key is valid
  3. Check file is .txt format
  4. Try smaller file first

PROBLEM: "Build failed on Vercel"
SOLUTION:
  1. Check all files are in correct folders
  2. Verify package.json has "next" and "openai"
  3. Check build logs in Vercel dashboard

PROBLEM: "API Key invalid"
SOLUTION:
  1. Go to https://platform.openai.com/api-keys
  2. Create a new key (old one may be inactive)
  3. Make sure key starts with "sk-"
  4. Update .env.local with new key
